## Principles to carry (from the user, this session)

- **Spec-first:** question whether a signal is needed; if yes, fix **spec Ôćĺ compute Ôćĺ tracing**,
  never patch the symptom downstream.
- **Traceable, not assumed:** every number traces to computed data, enforced deterministically.
  Fluent Ôëá correct.
- **Deterministic where mechanical** (distributions/thresholds/scans), **LLM where judgment**
  (scalar so-what) ÔÇö validated either way.
- **Detect data + dates carefully before ingestion** (older RBI formats vary: sheet names,
  row offsets, abbreviated months, bank renames/merges). The detector + extractor now share
  one robust path.
